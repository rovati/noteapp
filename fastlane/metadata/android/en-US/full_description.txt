<p><i>Notes</i> is a simple but effective app for taking plain text notes and making check lists. It started as a little playground to learn a new dev framework but quickly evolved into the desire to create a meaningful product.</p><p>Apps these days tend to offer an elaborate set of features, at the cost of asking the user for many permissions. <i>Notes</i> takes the opposite approach: it carefully picks a small set of essential features so that the user does not have to worry about security and privacy related issues. Your data is yours to keep.</p>
